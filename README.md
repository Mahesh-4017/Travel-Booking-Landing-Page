# 🏨 Booking UI Clone

🔗 Live Demo: https://travel-booking-landing.netlify.app

This is a **frontend project** inspired by Booking.com, created during my early learning phase of web development.

## 🚀 Live Features

* Modern and clean UI design
* Fully responsive layout (mobile + desktop)
* Hero section with search box
* Property categories (Hotels, Villas, Apartments, Resorts)
* Trending destinations section
* Promotional offer banner
* Footer with multiple sections

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox + Grid)
* Google Fonts

## 🎯 Purpose of This Project

This project was built to:

* Practice HTML structure and semantic elements
* Improve CSS layout skills (Grid & Flexbox)
* Learn responsive design
* Build real-world UI like Booking.com
